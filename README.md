# Simple-GAN-Base-on-Matlab
* simple generative adversarial networks base on matlab 
* 基于matlab上的简单生成对抗性网络
* 代码中有两个文件，gan_adam.m和gan_sgd.m

### gan_adam.m
推荐使用ADAM优化器对GAN网络进行更新
